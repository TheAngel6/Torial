What is territorial custom maps server about?

The Custom maps server is a server on which everyone can send their personalized maps, vote to show whether the map we like or not✅❌ and then sort them.
Everyone can then come to this server to test the maps or suggest some.

How do i support territorial custom maps server?

you can support us by sharing this server, by talking about it around you, by joining us, by publishing your maps if you are a map creator, and/or by testing our maps; critiquing the maps (good or bad with arguments to help everyone improve), and making suggestions to help us improve the server.
